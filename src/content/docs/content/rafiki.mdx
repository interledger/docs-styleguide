<Card title="Before committing changes" icon="star">

Your PR title must contain the appropriate prefix (probably `docs:`).


Run the following command in the `/packages/documentation` directory:

```
npx prettier . --write
```

</Card>

<Card title="After a Starlight upgrade" icon="star">
```
cd <to project folder>
pnpm install
cd packages/documentation
pnpm start
```
</Card>
