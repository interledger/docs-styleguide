title: StylishHeader

import { StylishHeader } from "@interledger/docs-design-system";

Use this component to exclude a heading from the page-level TOC and stylize the heading with an accent color.

To use it, your docs page must be in `.mdx` format. Please change the format from `.md` to `.mdx` if necessary. All your existing markdown will still be supported without issue.

## Usage

Import the `StylishHeader` component like so:

```jsx
import { StylishHeader } from "@interledger/docs-design-system";
```

Use the `<StylishHeader>` component within your content like so:

```jsx
<StylishHeader>Wow I'm a stylish header</StylishHeader>
```

## Working example

| Boring heading                                                  | Stylish heading                                          |
| --------------------------------------------------------------- | -------------------------------------------------------- |
| <p style="font-weight: bold; font-size: 20px;">I'm so basic</p> | <StylishHeader>Wow I'm a stylish heading</StylishHeader> |
