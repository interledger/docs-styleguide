Use the `img-outline` class to add a border to an image. The border is `2px solid var(--color-accent-transparent)` with rounded edges and a bit of a drop shadow. The `color-accent-transparent` is a hue of one of our doc system accent colors.

```html
<img
  src="/img/snippets/view-keys.png"
  alt="Alt text goes here"
  class="img-outline"
/>
```
